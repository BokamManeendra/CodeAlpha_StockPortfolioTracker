# 📈 Stock Portfolio Tracker

A simple **Stock Portfolio Tracker** developed using **Python** as part of the **CodeAlpha Python Programming Internship**.

## 📌 Project Overview

The Stock Portfolio Tracker allows users to enter stock names and quantities, then calculates the total investment value using predefined stock prices. The program also saves the investment summary to a text file.

---

## ✨ Features

- 📊 Predefined stock prices
- 📝 User input for stock names and quantities
- 💰 Calculates total investment value
- 📄 Saves the result to a text file
- 💻 Console-based application
- 🚀 Beginner-friendly Python project

---

## 🛠️ Technologies Used

- Dictionary
